## Getting Started

Using this theme is straightforward:

1. **Install the theme** in your Hugo site
2. **Configure your menus** in `hugo.toml`
3. **Add your content** to the `content` directory
4. **Customize colors** using CSS variables
5. **Deploy your site** and enjoy!

## Customization Options

The theme is built with **CSS custom properties** (variables) that make customization easy:

```css
:root {
  --color-primary: var(--color-blue-2);
  --font-size-base: 1.125rem;
  --space-4: 1rem;
}
```

You can override these values to match your brand colors and preferences.
